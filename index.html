This is a how to guide for using GitHub for beginners, done by a beginner.

By Nolan Porterfield

If you want a crash course on how to use GitHub and Netlify, then this is the list for you. Trust me.

  1. First open terminal
  2. Enter cd to allow changes to be made to your folder
  3. To add new files click git add
  4. The changes won't be seen, but they've been made
  5. Type "git add" to see changes
  6. Make the changes to the main branch using "git branch -M main"
  7. Connect a local foler to GitHub
  8. Push commits to GitHub using "git push origin main"
